# About this layout

A reference layout that shows how a gallery entry can carry richer documentation. The card shows the
short description; this longer write-up appears when you expand the card.

## What's inside

- An **embedded plugin page** and a **compact embeddable panel** from the
  [Flexible Layouts Example plugin](https://github.com/jaysuk/Flexible-Layouts).
- A demonstration of **dependency-aware import** — on import, Flexible Layouts checks that the required
  plugin is installed and warns you if it isn't.

## How to use it

1. Download the file with the **Download** button.
2. In DuetWebControl open **Flexible Layouts → Backup & share → Import**.
3. Pick the file. If a required plugin is missing you'll be prompted before anything changes.

## Make it your own

Use this as a starting point: duplicate the folder under `layouts/`, drop in your own export and
screenshots, and write a `details.md` like this one. Markdown supports headings, **bold**, *italic*,
`code`, lists and [links](https://github.com/jaysuk/flexible-layouts-gallery).
